# AI Smart Task Manager Dashboard

A modern frontend project built using **React, HTML, CSS, and JavaScript**. This application demonstrates component-based architecture, state management, responsive UI design, and browser data persistence using Local Storage.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Search tasks in real time
* Task priority levels (High, Medium, Low)
* Dashboard statistics
* Local Storage persistence
* Fully responsive design

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* React.js
* Vite

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/ai-smart-task-manager.git
cd ai-smart-task-manager
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```
